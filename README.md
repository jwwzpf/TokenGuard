# Token Guard

Stop feeding your whole repo to AI.

Token Guard is a local-first token efficiency layer for Claude Code and Codex. It reduces wasted tokens from oversized file reads, noisy terminal output, bloated handoffs, repeated commands, verbose agent replies, and fallback detours — without running a daemon or uploading code.

## Install in a project

```bash
token-guard install
```

## Daily commands

```bash
token-guard doctor
token-guard report
token-guard open-report
token-guard uninstall
```

## Trust model

- local-first
- no daemon
- no cloud backend
- no code upload
- visible project folder: `TokenGuard/`
- clean uninstall
