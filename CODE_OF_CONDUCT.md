# Code of Conduct

Token Guard is an open-source project for developers building with AI coding agents.

We expect everyone participating in this project to:

- be respectful
- assume good intent
- give constructive feedback
- avoid harassment, discrimination, or personal attacks
- keep technical disagreement focused on the work

Unacceptable behavior includes:

- insults or harassment
- discriminatory language
- threats
- publishing private information
- repeated off-topic disruption

Maintainers may remove comments, close issues, or block users who violate this code of conduct.
